Flask

NewPanel Linux管理面板

技术栈

  • 前端：Vue Vue-router Vue-cookies Vuex Axios Element-ui
  • 后端: Flask Flask-restful
  • Shell脚本

主要功能

  • 网站创建管理、FTP创建管理、数据库创建管理
  • 系统信息在线查看（CPU、内存、负载、带宽使用情况、磁盘使用率）
  • 在线安装软件、修改密码、文件管理

网站管理

  • 输入站点名一键创建网站、FTP、数据库
  • 支持部署https
  • 支持配置301，伪静态
  • 支持调整额外参数 404错误页、安全加固、防盗链等

FTP管理

  • 增删改查FTP站点
  • 开关FTP站点

数据库管理

  • 一键创建数据库
  • 一键开关远程连接

软件管理

  • 一键安装或卸载Nginx、Php、Mysql、Ftp

文件管理

  • 在线查看文件
  • 编辑文件
  • 删除文件
  • 新建文件

安装方式

wget http://panel.ropon.top/panel/lnmp.tar.gz && tar xf lnmp.tar.gz && cd lnmp && ./install.sh

源码获取

wget http://panel.ropon.top/panel/lnmp/panel-1.1.tar.gz

常见命令

  • 启动关闭重启面板服务 systemctl startstoprestart newpanel
  • 重置管理用户名及密码： source /home/panel/newpanel/.py3env/bin/activate && cd /home/panel/newpanel && python3 manager.py repasswd
